
# Doc2PDF

A web app for convert .docx files to pdf format


![Logo](https://github.com/alxandrkgb/doc2pdf/blob/master/public/favicon-32x32.png)


## Features

- Light/dark mode
- Live previews
- Fullscreen mode
- Cross platform
